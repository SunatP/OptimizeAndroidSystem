# Screen Setting

```bash
# Need su or super user before change the screen size
wm size 1080x1920 # for FHD screen
wm size 1440x2560 # for 2K screen
wm size reset

wm size
Physical size: 1440x2560 # Default Value
Override size: 1080x1920

wm density
Physical density: 560 # Default Value
Override density: 448
```
